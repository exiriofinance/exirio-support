User interaction needed error

When connecting an external Institution to Exirio, you may occasionally encounter an error message stating "USER_INTERACTION_NEEDED". This is a common issue that can be easily resolved.

What does this error mean?

The "USER_INTERACTION_NEEDED" error typically indicates that the account has some pending checks/tasks that need the User to access the official website and complete them.

How to resolve this error

  1. Complete pending tasks
    • Access the official website of the external Insitution and complete any pending tasks.
  2. Try connecting the Institution to Exirio again
    • Go back to Exirio and try connecting the Institution again.

Important note

If you've completed the steps above and you're still experiencing this error, please contact our support team at support@exirio.com. We'll be happy to help investigate the issue further.
